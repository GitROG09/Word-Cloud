# 🧠 NLP Word Cloud with Sentiment Analysis

This project is a Streamlit-based NLP application that:

- Generates word clouds from text or PDF input
- Classifies words as Positive or Negative using NLTK lexicon
- Highlights positive words (green) and negative words (red)

## 🚀 Features
- Text input support
- PDF upload support
- Sentiment classification
- Word cloud visualization

## 🛠️ Technologies Used
- Python
- Streamlit
- NLTK
- WordCloud
- PyPDF2

## ▶️ How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
